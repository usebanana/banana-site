# banana-site

Marketing and legal pages for [Banana](https://usebanana.app) — a shared to-do list app for iOS.

## Structure

- `index.html` — marketing landing page
- `privacy.html` — privacy policy
- `terms.html` — terms of service
- `styles.css` — shared styles

## Deployment

Auto-deployed to Cloudflare Pages from the `main` branch on every push.
